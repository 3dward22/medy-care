# Product Backlog (Medy-Care)

This backlog lists prioritized product work for the Medy-Care system based on currently implemented modules and near-term improvements.

| ID | Epic | User Story | Priority | Status |
|---|---|---|---|---|
| PB-01 | Authentication & Access | As a user, I want to securely log in so that I can access my role-based dashboard. | High | Done |
| PB-02 | Authentication & Access | As an admin, I want account verification for staff users so that only approved personnel can access operations. | High | Done |
| PB-03 | User Management | As an admin, I want to manage users so that I can maintain active and valid accounts. | High | Done |
| PB-04 | Appointment Booking | As a student, I want to book, view, and cancel appointments so that I can request clinic support easily. | High | Done |
| PB-05 | Appointment Workflow | As a nurse, I want to start, complete, or decline appointment sessions so that patient flow is tracked correctly. | High | Done |
| PB-06 | Admin Appointment Oversight | As an admin, I want to view all/today/week appointments so that I can monitor clinic activity. | Medium | Done |
| PB-07 | Student Records | As a nurse, I want to view student medical records so that treatment decisions are informed. | High | Done |
| PB-08 | Emergency Records | As a nurse, I want to create and review emergency records so that urgent cases are documented properly. | High | Done |
| PB-09 | Notifications | As a user, I want in-app notifications and read tracking so that I stay updated on appointment activity. | Medium | Done |
| PB-10 | Guardian Communication | As clinic staff, I want to send guardian SMS notifications so that guardians are informed when needed. | High | Done |
| PB-11 | Reporting | As an admin, I want monthly reports so that I can review service trends and outcomes. | Medium | Done |
| PB-12 | Appointment History Enhancements | As an admin, I want exportable appointment history with filters so that audits and reviews are easier. | Medium | Planned |
| PB-13 | Notification Preferences | As a user, I want notification channel preferences so that I receive alerts in my preferred way. | Medium | Planned |
| PB-14 | Security Hardening | As a system owner, I want stronger session/security controls so that the platform remains protected. | High | Planned |
| PB-15 | Analytics Dashboard | As management, I want KPI cards and trend charts so that performance can be tracked quickly. | Medium | Planned |

## Backlog Prioritization Notes

- **Highest priority:** appointment lifecycle reliability, user approval flow, and emergency documentation.
- **Next priority:** reporting depth, notification preferences, and analytics visibility.
- **Ongoing:** security hardening and operational quality improvements.
